About this api

User Routes

POST Sign up -> /api/v1/user/signUp

POST Sign in -> /api/v1/user/signIn

POST Log out -> /api/v1/user/signOut


Blog Routes

GET View all blogs -> /api/v1/blog

GET View blog -> /api/v1/blog/{id}

POST Create blog -> /api/v1/blog

PUT Update blogs -> /api/v1/blog/{id}

DELETE Delete blog -> /api/v1/blog/{id}


Setup
  1. Clone git repo
  2. Create .env file and edit the ff. postgres config
    • DB_HOST
    • DB_PORT
    • DB_PASSWORD
    • DB_NAME=blog
  3. Download node
  4. run "npm install" in project path in cmd
  5. run "npm start" in project path in cmd
  6. Server should start!!
